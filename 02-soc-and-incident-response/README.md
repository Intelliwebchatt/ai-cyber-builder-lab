# SOC and incident response

This track will demonstrate monitoring, detection, triage, containment, recovery, and lessons learned in an authorized lab.

## Planned projects

- Windows and Linux log collection
- mini-SOC using a free SIEM or security-monitoring platform
- alert triage and escalation records
- Sigma-style detection logic and test events
- MITRE ATT&CK coverage map
- incident containment and recovery exercise

**Status:** Planned. Mission 001 provides the first incident-analysis dataset.
