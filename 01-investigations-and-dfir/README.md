# Investigations and digital forensics

This track develops the technical evidence needed for cybercrime, digital-forensics, insider-threat, and incident-investigation roles.

## Planned evidence

- structured case files and evidence logs
- integrity verification with cryptographic hashes
- host and account timelines
- disk, browser, log, and memory artifact analysis
- defensible findings with confidence statements
- technical and executive reporting
