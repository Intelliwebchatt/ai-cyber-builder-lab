# Operational technology and critical infrastructure

This track combines physical-systems knowledge, safety, reliability, and cybersecurity. Labs will use simulations and isolated software rather than live industrial equipment.

## Planned projects

- building-automation and backup-power asset inventory
- Purdue-style network segmentation diagram
- safe Modbus or OpenPLC simulation
- OT threat model and consequence analysis
- monitoring and anomaly-detection plan
- safety-aware incident response and recovery exercise

**Status:** Planned. Networking foundations and official introductory ICS training come first.
